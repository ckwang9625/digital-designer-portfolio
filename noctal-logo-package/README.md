# Noctal Logo Package

Sample brand identity package for Noctal, a premium dissolvable sleep-strip supplement brand.

## Contents

- `noctal-logo-packages/` - working asset folders for three logo concepts.
- `noctal-logo-packages.zip` - compressed package for sharing.
- `create_noctal_logo_packages.mjs` - generator script used to create the sample SVG/PDF source files.

## Concepts

- `concept-01-switch` - switch-off / power-down mark.
- `concept-02-dissolve` - dissolvable oral strip mark.
- `concept-03-neural-calm` - calm signal / mental quiet mark.

Each concept includes dark and light primary logos, transparent lockups, favicon/icon files, PNG exports, and one-page guidelines in SVG/PNG/PDF.
